# 🚨 Women Safety Analysis using EDA

📊 Exploratory Data Analysis | 📈 Trends | 🔍 Insights

---

## 📌 Overview
This project performs **Exploratory Data Analysis (EDA)** on crimes against women in India to identify patterns, trends, and key insights using real-world data.

---

## 🎯 Objectives
- Analyze different types of crimes against women
- Identify states with the highest crime rates
- Study trends over time
- Understand relationships between different crime categories

---

## 📂 Dataset
The dataset includes crime records across various states and years with features such as:

- State  
- Year  
- Rape  
- Kidnapping & Abduction (K&A)  
- Dowry Death (DD)  
- Assault on Women (AoW)  
- Assault on Minors (AoM)  
- Domestic Violence (DV)  
- Women Trafficking (WT)  

---

## 🧹 Data Preprocessing
- Removed unnecessary columns  
- Checked and handled missing values  
- Cleaned inconsistent state names  
- Prepared dataset for analysis  

---

## 📊 Analysis & Visualizations

### 🔹 Top States with Highest Crimes
States like Uttar Pradesh, Madhya Pradesh, and West Bengal show higher crime counts.

### 🔹 Crime Distribution
Domestic Violence contributes the largest share, followed by Assault on Women and Kidnapping.

### 🔹 Correlation Analysis
Strong correlation observed between Domestic Violence and Total Crimes.

### 🔹 Year-wise Trends
Overall increase in crimes with a noticeable rise after 2012 and slight variations in recent years.

---

## 📈 Key Insights
- Domestic Violence is the most dominant crime category  
- Crimes have increased over time  
- Certain states consistently report higher cases  
- Crime categories are interrelated
